# InferentialMistakes

#update 2026
Skill.md file: This Claude Code skill provides an automated statistical quality checklist for movement science, motor control, and neuroscience research. It triggers automatically whenever you are analysing data, writing a results section, or reviewing analysis code in Claude Code CLI.
The checklist is based on Makin & Orban de Xivry (2019, eLife, doi:10.7554/eLife.48175) and covers the 10 most common statistical mistakes found in the neuroscience and movement science literature: missing control conditions, indirect group comparisons, inflated units of analysis, spurious correlations, underpowered designs, circular analysis, p-hacking, uncorrected multiple comparisons, over-interpreted null results, and correlation/causation confusion.
Beyond the original paper, the skill includes kinematics-specific guidance on time-series analysis (SPM1D, permutation tests), bilateral limb data, trial-level aggregation, and pre-post experimental designs. Tool-specific advice is provided for both MATLAB and Python (scipy, pingouin) users.
